# Area
Это проект вёрстки лендинга по макету из Figma Community под названием Area.

Посмотреть работающий сайт можно перейдя по этой [ссылке](https://jlyaguxa.github.io/Area/)


Проект был реализован с помощью таких языков и ПО как: HTML, CSS, JS, Figma

Для локального запуска достаточно склонировать репозиторий:
```bash
   git clone https://github.com/jlyaguxa/Area.git
```
После чего открыть файл index.html в папке проекта

Вёрстка была выполнена по [макету](https://www.figma.com/community/file/1487309170684591074/modern-product-launch) человека из Figma Community
